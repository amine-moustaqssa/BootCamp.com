Qu'est-ce qu'un « bootcamp » ?

Le Bootcamp est une mode de formation intensive et accélérée de trois à six mois qui simule les conditions de travail dans le milieu professionnel afin de vous permettre de développer simultanément et de manière efficace toutes les zones de compétence requises pour réussir dans les métiers du digital : compétences techniques, soft-skills, méthodologie et éthique de travail, culture métier, capacité à s’auto-former… et ce en se basant en grande partie sur de la pratique travers des projets des études de cas pratiques.

Learn React

React est une bibliothèque JavaScript libre développée par Facebook depuis 2013. Le but principal de cette bibliothèque est de faciliter la création d'application web monopage, via la création de composants dépendant d'un état et générant une page HTML à chaque changement d'état.

Nos filieres

Le développement web frontal correspond aux productions HTML, CSS et JavaScript d’une page internet ou d’une application qu’un utilisateur peut voir et avec lesquelles il peut interagir directement.

WordPress est un système de gestion de contenu gratuit, libre et open-source. Ce logiciel écrit en PHP repose sur une base de données MySQL et est distribué par la fondation WordPress.org

Un développeur full stack, appelé aussi développeur généraliste, développeur polyvalent ou plus familièrement « développeur à tout faire », est un codeur capable de réaliser la programmation d’un site ou d'une application web à la fois en front-end et back-end.
